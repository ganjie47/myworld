数据查询器 Query Builder

数据查询器 重新设计的公安数据查询器能让用户在全部数据集中进行查询，自由组合、修改条件，查询结果可生成各种图表的报表。相较原来查询器的偏专业和不易操作，新版查询器能更好引导用户查询 - 分步骤进行选图表、选择字段、输入条件操作。 设计要点： 1. 在了解业务场景需求后设计分步操作
2. 字段条件修改和数据结果浏览同屏的设计让用户在生成报表前对查询结果进行调整，操作起来更灵活便捷
Query Builder The redesigned query builder for police cloud allows users to make queries through all data sets, freely combine and modify conditions, query results can be generated into various types of charts. Compared with the original query builder that was more for professional personnel, the new query builder is designed for wider range of users to query with separated steps of selecting charts, choosing fields, inputting conditions. Design Keypoints: 1. Designed step-by-step operations based on the requirements of business scenarios
2. The design of field condition modification and data result browsing on the same screen allows users to adjust the query results before generating final reports, which is more flexible and convenient to use
